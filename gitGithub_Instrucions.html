Instruções de comandos no git para github

git init

Serve para criar um repositório git na pasta escolhida

git add .

Serve para enviar seus arquivos para uma área temporária chamada "Stage"

git commit -m "Texto Explicativo"

Serve para salvar uma nova versão no repositorio git

git branch -m main

Serve para auterar o branch master para main no git, padrão no github, somente uma vez.

git remote add origin git@github.com: /.git

Serve para associa seu projeto no github com o git (SSH)

git push -u origin main

Serve para adicionar repositorio no github, observação este comando é somente para primeira vez.

Para os demais atualizações de repositorios utiliza se somente "git push"
